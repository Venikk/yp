# Итоговый проект курса. 
## В этом проекте необходимо было в первую очередь сориентироваться, какие методы из всего курса необходимо применить. Задача более-менее стандартная. Телеком компания, пытающаяся предсказать отток клиентов и предложить им скидки на услуги. Не предложишь потенциально недовольным – уйдут, предложишь лояльным – теряешь деньги.
