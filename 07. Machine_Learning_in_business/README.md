# Проект по теме "Машинное обучение в бизнесе"
## В этом проекте необходимо было выбрать модель, которая бы обучилась на имеющемся датасете с историческими данными о параметрах нефтяных скважин, а после обучения сформировала бы прогноз по региону, в котором наиболее выгодно было бы вести разработку месторождений.
